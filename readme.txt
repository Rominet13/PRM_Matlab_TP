Création de trajectoire pour plusieurs robots
Méthode PRM ( Probabilistique RoadMap)
(TP de synthése de mouvement/chemin)

Ensemble de fichiers matlab implémentant la technque PRM pour création de trajectoire à plusieurs robots.
Interface graphique animé pour voir l'évolution des mobiles cylindriques
Etape: 
	- création du graphes des positions (étape la plus couteuse en temps)
	- résolution via l'algorithme A*
	- affichage du résultat (animation)

	+ Archives dans 'sauvegarde' pour directement lancer les animations sur d'anciens tests.
